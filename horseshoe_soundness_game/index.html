Phase 2: The Horseshoe

Now we execute the Horseshoe Method. We use script metavariables like 𝒜 and ℬ to stand in for arbitrary sentences, since this proof must apply generally to any valid application of the rule.

  1. Start on Bottom Left (Assumption): Consider an arbitrary valuation v satisfying δₙ
  2. Up (By Rule Syntax): Inference about how v relates to δᵢ (via subset facts)
  3. Across (By Inductive Hypothesis): Inference about how v relates to φᵢ (via the Inductive Hypothesis)
  4. Down: Inference that v satisfies φₙ
